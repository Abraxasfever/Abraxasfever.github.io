漏洞利用-rpcbind

2020-09-07 HACK学习

rpcbind介绍

通俗来说，rpcbind是NFS中用来进行消息通知的服务。
一般情况下rpcbind运行在111端口。并且NFS配置开启rpcbind_enable="YES"

探测目标rpcbind

namp -sV -p 111 IP地址 探测rpcbind版本信息
searchsploit rpcbind 查看可利用的漏洞

namp脚本探测

在nmap中使用nmap -p 111 --script=rpcinfo 目标地址ip 来探测目标的rpcinfo信息

metasploit模块探测

使用metasploit下的auxiliary/scanner/misc/sunrpc_portmapper 进行目标探测。
use auciliary/scanner/misc/sunrpc_portmapper
