VMware Workstation

虚拟机，指的是通过软件模拟的具有完整硬件西塘功能的、运行在一个完全隔离环境中的完整计算机系统
Vmware Workstation 功能强大的桌面虚拟计算机软件，提供用户可在单一的桌面上同时运行 不同的操作系统和进行开发、测试、部署新的应用程序的最佳解决方案。
http：//www.vmware.com/

Vmware中的网络分类

Vmware中具有四种不同的网络模式。其中自定义模式更具有扩展性，方便构建自定义网络。

桥接模式：

比较常用的一种网络模式，用来设置虚拟机与主机具有同等地位，前提是需要在动态分配Ip地址网路中，如果在静态分配Ip地址的网络中，需要手动设置对应的IP地址.

NAT模式（VMnet8）：

NAT网络地址转换，主机与虚拟机构成一个网络，虚拟机可以与主机以外的网络通信，但是主机以外的网络无法与虚拟机通信。

仅主机模式（VMnet1）：

虚拟机完全封闭，不与外部网络通信，但是可以与当前主机进行通信。

Kali linux

基于Debian的Linux发行版本，设计用于安全检测操作系统，内部集成了600+安全工具。
官网下载
更新kali

apt update && apt upgrade && apt dist-upgrade

kali2020

默认用户由root改为kali，于是，很多命令无法直接使用，在设置共享文件夹以及界面语言方面都有了不小的难度，在经历了安装搜狗输入法失败后，成功安装了谷歌输入法
汉化：

cd /etc/apt/
sodu vim sources.list
#中科大的源
    deb http://mirrors.ustc.edu.cn/kali kali-rolling main non-free contrib
    deb http://mirrors.ustc.edu.cn/kali kali-rolling main contrib non-free
    deb-src http://mirrors.ustc.edu.cn/kali kali-rolling main contrib non-free
    deb http://mirrors.ustc.edu.cn/kali-security kali-current/updates main contrib non-free
    deb-src http://mirrors.ustc.edu.cn/kali-security kali-current/updates  main contrib non-free
sudo apt-get update
sudo apt-get install ttf-wqy-microhei ttf-wqy-zenhei xfonts-wqy fonts-wqy-zenhei fonts-wqy-microhei
sudo dpkg-reconfigure locales
空格选中en_US.UTF-8 UTF-8和zh_CN.UTF-8 UTF-8以及zh_CN.GBK GBK
然后默认zh_CN.UTF-8
sudo reboot

安装谷歌输入法：

sudo apt-get update && apt-get upgrade
sudo apt-get install fcitx
sudo apt-get install fcitx-googlepinyin
sudo reboot

然后就可以使用CTRL+SPACE切换输入法，输入中文了，感谢万能的网络！
